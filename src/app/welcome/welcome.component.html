Welcome to Dockyard

The SPA will have the following features:

  • A page that lists all boats
  • Clicking a boat in above list will show a page having all available details about a specific boat, including the names of the workers who are working on that boat
  • UI to do CRUD operations on the boat data set
  • UI to assign / un-assign workers to boats - there is no need to build a CRUD UI for the workers data set.
